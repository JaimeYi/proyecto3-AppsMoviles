Descripción aplicación a desarrollar en proyecto 3: 
Contexto: aplicación para registrar los gastos personales. 
Requisitos: 
• La aplicación permitirá el ingreso de una transacción como gasto. 
• La aplicación permitirá el ingreso de una transacción como ingreso. 
• La aplicación mostrará el balance total. 
• La aplicación mostrará el total de ingresos. 
• La aplicación mostrará el total de gastos. 
• La aplicación mostrará el listado de transacciones. 
 
Restricciones: 
• Se debe aplicar el patrón MVVM, con Community Toolkit MVVM. 
• Se debe utilizar SQLite para persistir las transacciones realizadas.
